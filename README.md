# CoffeeAndToffee

Welcome to **CoffeeAndToffee**!  
An interactive web project crafted with JavaScript, CSS, and HTML.

## 🚀 Live Demo

Check out the live version of the site here:  
[https://coffee-and-toffee-three.vercel.app/](https://coffee-and-toffee-three.vercel.app/)

## 🛠️ Built With

- **JavaScript** (95%) — Core logic and interactivity
- **CSS** (4.7%) — Stylish and responsive design
- **HTML** (0.3%) — Markup structure

## 📦 Features

- Beautiful, user-friendly interface
- Responsive design for desktop and mobile
- Smooth and interactive user experience

## 🌐 Getting Started

To run your own local copy:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/AdityaGadbail/CoffeeAndToffee.git
   ```
2. **Open the project:**
   - Use any web server to serve the files (e.g. VSCode Live Server extension or Python’s SimpleHTTPServer).
   - Or, simply open `index.html` in your browser if no server is required.
